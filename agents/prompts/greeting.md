# Greetings

Two lines live here. Mia uses the first one when no brokerage name was typed on /demo, and the second when someone DID type a brokerage name.

## Default greeting (no brokerage name typed)

Sunbelt Realty, this is Mia — how can I help you today?

## Personalized greeting (when a brokerage name IS typed)

Use `{name}` as a placeholder. It gets replaced with whatever the user types on the /demo page.

{name}, this is Mia — how can I help you today?
